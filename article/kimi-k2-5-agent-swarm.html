Kimi K2.5 เปิดตัว Agent Swarm 100 Sub-Agent ทำงานขนาน พร้อม 1,500 Tool Calls

2026-05-01 · ~1 นาที · Industry

Moonshot AI บริษัท AI จากจีน เปิดตัว Kimi K2.5 โมเดล open-source multimodal ที่มีฟีเจอร์พิเศษที่โลกยังไม่เคยเห็นมาก่อน นั่นคือ Agent Swarm

เกิดอะไรขึ้น?

Agent Swarm คือระบบที่สามารถแตกงานซับซ้อนออกเป็น subtask แล้วสร้าง sub-agents สูงสุด 100 ตัว ทำงานขนานกันได้ทันที โดยไม่ต้องกำหนด role หรือ workflow ล่วงหน้า ระบบ orchestrator จะจัดสรรบทบาทเองทั้งหมด จุดเด่นของ Agent Swarm: สูงสุด 100 sub-agents ทำงานขนานกัน รองรับสูงสุด 1,500 tool calls ต่อ task เดียว sub-agents จะรับบทบาทอัตโนมัติ เช่น Researcher, Analyst, Fact Checker, Coder เป็น native multimodal model รองรับทั้ง coding และ vision

ทำไมถึงสำคัญ?

ปกติเวลาเราใช้ AI agent ทำงานซับซ้อน เราต้องกำหนด workflow เองว่า agent ไหนทำอะไร ต่อกันอย่างไร แต่ Agent Swarm ของ Kimi ทำลายข้อจำกัดนี้ แค่ให้ task ใหญ่ แล้วระบบจะแตกงาน สร้างทีม และดำเนินการเองทั้งหมด นี่คือการเปลี่ยนแปลง paradigm จาก one agent does everything เป็น orchestrator manages a team ซึ่งใกล้เคียงกับวิธีที่มนุษย์ทำงานกันจริงมากกว่า ยิ่งกว่านั้น Kimi K2.5 เป็น open-source ทำให้นักพัฒนาทั่วโลกสามารถนำไปใช้ ปรับแต่ง และต่อยอดได้

ต่อไปจะเป็นอย่างไร?

ถ้า Agent Swarm ทำได้จริงตามที่อ้าง จะเป็นจุดเปลี่ยนสำคัญของวงการ multi-agent AI การทำวิจัยขนาดใหญ่ การวิเคราะห์ข้อมูลซับซ้อน หรือการพัฒนาซอฟต์แวร์ จะเร็วขึ้นและมีประสิทธิภาพมากขึ้นอย่างที่ไม่เคยเป็นมาก่อน Source: Kimi Blog — Agent Swarm
