Claude Code บนเดสก์ท็อปได้รับการออกแบบใหม่ รองรับการทำงานแบบขนาน

2026-05-13 · ~1 นาที · Industry

วันนี้เรามีข่าวดีสำหรับนักพัฒนาทุกคน เพราะ Claude Code บนเดสก์ท็อปได้รับการออกแบบใหม่ทั้งหมดเพื่อช่วยให้คุณสามารถรันงานหลายๆ อย่างพร้อมกันได้ ไม่ต้องรออีกต่อไป!

ประสบการณ์ใหม่บนเดสก์ท็อปที่เปลี่ยนไป

สำหรับนักพัฒนาหลายคน วิธีการทำงานกับ AI ได้เปลี่ยนไป คุณไม่ได้พิมพ์คำสั่งเดียวแล้วรอผลอีกต่อไปแล้ว แต่ตอนนี้คุณสามารถทำงานหลายอย่างพร้อมกัน: แก้โครงสร้างโปรเจกต์นึง แก้ไขบักในอีกโปรเจกต์ และเขียนเทสในโปรเจกต์ที่สาม แล้วตรวจสอบผลลัพธ์ของแต่ละอย่างขณะที่งานกำลังทำงานอยู่ แอปใหม่นี้ถูกสร้างขึ้นมาเพื่อตอบสนองวิธีการทำงานแบบนี้โดยเฉพาะ: หลายอย่างทำงานพร้อมกัน และคุณเป็นผู้ควบคุมทั้งหมด

รันเซสชันพร้อมกันได้ด้วยซิดบาร์ใหม่

ซิดบาร์ใหม่จะรวบรวมเซสชันที่กำลังทำงานและเซสชันล่าสุดทั้งหมดไว้ในที่เดียว คุณสามารถเริ่มงานในหลายๆ โปรเจกต์พร้อมกัน แล้วสลับไปมาระหว่างพวกมันเมื่อได้ผลลัพธ์ คุณสามารถกรองตามสถานะ โปรเจกต์ หรือสภาพแวดล้อม หรือจัดกลุ่มซิดบาร์ตามโปรเจกต์เพื่อค้นหาและดำเนินการเซสชันต่อได้เร็วขึ้น เมื่อเซสชันใดๆ ถูกปิดหรือผสานเข้ากับ PR มันจะถูกเก็บไว้โดยอัตโนมัติ เพื่อให้ซิดบาร์ของคุณโฟกัสเฉพาะงานที่ยังคงทำงานอยู่เท่านั้น และเมื่อคุณต้องการถามคำถามระหว่างทำงาน คุณสามารถเปิดแชทข้างๆ (⌘ + ; หรือ Ctrl + ;) เพื่อแยกสนทนาออกไปได้ แชทข้างๆ จะดึงบริบทจากเธรดหลัก แต่จะไม่เพิ่มอะไรกลับเข้าไปในเธรด เพื่อหลีกเลี่ยงการทำให้งานของคุณเสียทิศทาง

ตรวจสอบและส่งงานได้โดยไม่ต้องออกจากแอป

การออกแบบใหม่นี้นำเครื่องมือที่ใช้บ่อยๆ เข้ามาไว้ในแอป ช่วยให้คุณสามารถตรวจสอบ ปรับแต่ง และส่งงานของ Claude ได้โดยไม่ต้องกระโดดไปยังเอดิเตอร์อื่น: เทอร์มินัลแบบบูรณาการ: รันเทสหรือบิลด์ข้างๆ เซสชันของคุณ เอดิเตอร์ไฟล์ในแอป: เปิดไฟล์ แก้ไขโดยตรง และบันทึกการเปลี่ยนแปลง ตัวดู Diff ที่เร็วขึ้น: สร้างขึ้นใหม่สำหรับประสิทธิภาพที่ดีขึ้นกับการเปลี่ยนแปลงขนาดใหญ่ ตัวแสดงผลขยาย: เปิดไฟล์ HTML หรือ PDF ในแอป นอกเหนือจากรันแอปเซิร์ฟเวอร์ในเครื่องในแผงดูตัวอย่าง ทุกแผงสามารถลากและวางได้ จัดเรียงเทอร์มินัล ตัวอย่าง ตัวดู Diff และแชทในรูปแบบกริดที่ตรงกับวิธีการทำงานของคุณ

รองรับเทคโนโลยีที่คุณใช้

แอปเดสก์ท็อปตอนนี้มีฟีเจอร์เทียบเท่ากับปลั๊กอิน CLI ถ้าองค์กรของคุณจัดการปลั๊กอิน Claude Code แบบรวมศูนย์ หรือคุณติดตั้งปลั๊กอินของตัวเองในเครื่อง พวกมันจะทำงานในแอปเดสก์ท็อปได้เหมือนกับในเทอร์มินัลทุกประการ คุณยังคงสามารถรันเซสชันในเครื่องหรือบนคลาวด์ได้เหมือนเดิม การรองรับ SSH ขณะนี้ขยายไปยัง Mac ควบคู่กับ Linux ช่วยให้คุณสามารถเชื่อมต่อเซสชันกับเครื่องระยะไกลจากทั้งสองแพลตฟอร์มได้

ปรับแต่งตามสไตล์การทำงานของคุณ

โหมดการดูสามแบบ—Verbose, Normal และ Summary—ช่วยให้คุณปรับอินเตอร์เฟซจากการแสดงผลการเรียกใช้เครื่องมือของ Claude แบบเต็มรูปแบบไปจนถึงเฉพาะผลลัพธ์ ทางลัดแป้นพิมพ์ใหม่ครอบคลุมการสลับเซสชัน การสร้างเซสชันใหม่ และการนำทาง กด ⌘ + / (หรือ Ctrl + /) เพื่อดูรายการทั้งหมด ปุ่มการใช้งานใหม่แสดงทั้งหน้าต่างบริบทและการใช้งานเซสชันของคุณในภาพรวมเดียว ภายใต้ประทุน แอปได้ถูกสร้างขึ้นใหม่เพื่อความเสถียรและความเร็ว และตอนนี้สามารถสตรีมการตอบกลับขณะที่ Claude กำลังสร้างผลลัพธ์ได้

เริ่มต้นใช้งานอย่างไร

แอปเดสก์ท็อปที่ออกแบบใหม่นี้พร้อมใช้งานแล้วสำหรับผู้ใช้ Claude Code ทุกแผนใน Pro, Max, Team และ Enterprise รวมถึงผ่าน Claude API ดาวน์โหลดแอป หรืออัปเดตและรีสตาร์ทถ้าคุณมีแอปอยู่แล้ว สำรวจเอกสารประกอบเพื่อเรียนรู้เพิ่มเติม การออกแบบใหม่นี้ไม่ได้เป็นเพียงการปรับปรุงหน้าตา แต่เป็นการปฏิวัติวิธีที่นักพัฒนาทำงานกับ Claude Code ด้วยความสามารถในการทำงานหลายอย่างพร้อมกัน คุณจะสามารถทำงานได้อย่างมีประสิทธิภาพมากขึ้น ลดเวลาที่ต้องรอ และมองเห็นภาพรวมของทุกโปรเจกต์ที่คุณกำลังทำงานอยู่ได้อย่างชัดเจน ลองดูว่ามันจะเปลี่ยนวิธีการทำงานของคุณได้ขนาดไหน แหล่งอ้างอิง: https://claude.com/blog/claude-code-desktop-redesign
